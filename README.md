# 儿时游戏

记录祖辈、父辈以及我们这一代儿时玩过的游戏。

## 计划

+ 从最简单的开始，初步只描述游戏规则；
+ 适当配上较好的图片；
+ 阅读查阅相关书籍找到更详细的描述.
